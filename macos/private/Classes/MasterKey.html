  • init(bytes:)

    Initialize a master key from a given byte array. he byte array is copied to a secure location and overwritten with zeroes to avoid the key being compromised in memory.

    Warning

    Do not initialize new keys with this function. If you need a new key, use init?() instead. This initializer is only to restore secret keys that were persisted.
